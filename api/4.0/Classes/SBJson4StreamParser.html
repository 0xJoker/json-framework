Overview

Low-level Stream parser

You most likely want to use the SBJson4Parser instead, but if you really need low-level access to tokens one-by-one you can use this class.

Properties

delegate

Delegate to receive messages

@property (nonatomic, weak) id<> delegate

Discussion

The object set here receives a series of messages as the parser breaks down the JSON stream into valid tokens.

Usually this should be an instance of SBJson4Parser, but you can substitute your own implementation of the SBJson4StreamParserDelegate protocol if you need to.

An NSData object containing the next chunk of JSON

@return – SBJson4ParserComplete if a full document was found – SBJson4ParserWaitingForData if a partial document was found and more data is required to complete it – SBJson4ParserError if an error occured.

Discussion

The JSON is assumed to be UTF8 encoded. This can be a full JSON document, or a part of one.
